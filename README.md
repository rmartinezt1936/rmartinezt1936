## Proyectos Técnicos

###  [Consumo de API REST con Python](https://github.com/rmartinezt1936/Consumo-API-Rest)
> Script en Python que realiza una solicitud `GET` a una API pública, procesa datos en JSON y los muestra en consola. Utiliza la librería `requests` y conceptos de manejo de datos estructurados.

###  [Consumo de API REST con JavaScript](https://github.com/rmartinezt1936/consumo-api-javascript)
> Página web simple que obtiene datos desde una API pública usando `fetch` en JavaScript, y los presenta de forma visual en tarjetas dinámicas. Publicado con GitHub Pages.

###  [Base de datos de empleados con SQL](https://github.com/rmartinezt1936/sql_empleados)
> Simulación de base de datos relacional con empleados y departamentos. Incluye creación de tablas, inserción de datos, consultas con `JOIN`, filtros y agrupaciones (`GROUP BY`).
